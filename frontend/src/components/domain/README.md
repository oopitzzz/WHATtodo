# 도메인 컴포넌트

비즈니스 로직과 밀접한 기능별 컴포넌트들입니다.

## 하위 디렉토리

- `todo/` - 할일 관련 컴포넌트
  - `TodoItem.jsx` - 할일 아이템 표시
  - `TodoList.jsx` - 할일 목록
  - `TodoForm.jsx` - 할일 생성/수정 폼

- `calendar/` - 캘린더 관련 컴포넌트
  - `CalendarView.jsx` - 캘린더 뷰

- `auth/` - 인증 관련 컴포넌트
  - `LoginForm.jsx` - 로그인 폼
  - `SignupForm.jsx` - 회원가입 폼
