## 🔍 API 엔드포인트 검증

SC-001 시나리오에서 다음 API 엔드포인트가 정상 작동함을 확인:

### 인증 API
- ✅ `POST /api/auth/signup` - 회원가입
  - 요청: `{ email, password, nickname }`
  - 응답: JWT 토큰 발급 및 사용자 정보 반환
  - 자동 로그인 처리

### 할일 API
- ✅ `GET /api/todos` - 할일 목록 조회
  - 필터링 및 정렬 기능 작동
  - 빈 상태 메시지 표시

- ✅ `POST /api/todos` - 할일 생성
  - 요청: `{ title, priority, dueDate }`
  - 응답: 생성된 할일 정보 반환
  - 즉시 목록에 반영

- ✅ `PATCH /api/todos/:id/complete` - 할일 완료
  - 상태를 ACTIVE → COMPLETED로 변경
  - 즉시 UI 업데이트
