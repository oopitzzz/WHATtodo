## 🎯 3. 핵심 기능 및 우선순위

### 3.1 MVP (최소 기능 제품) 범위
```
첫 출시(v1.0)에 반드시 포함되어야 할 기능:
- [할일 생성/수정/완료/삭제/복원 ]
- [ 우선순위 설정]
- [ ]

```
### 3.2 성공지표 KPI
```
사용자 성장 
- 3개월 내 목표 가입자 수 500명

사용자 참여도
- 목표 할일 참여도 50%

```


### 3.2 기능 우선순위 (MoSCoW)

#### Must Have (필수)
```
- [x ] 회원가입/로그인
- [ x] 할일 CRUD
- [ x] 할일 조회/추가/수정/완료/삭제/복원
- [x]  휴지통/할일 조회
```

#### Should Have (중요)
```
- [ x] 마감일 알림
- [ x] 우선순위 설정
- [ x] 모바일 지원 반응형 디자인 
- [x] 할일 정렬(우선순위/날짜)/필터링(날짜별/상태별)
```

#### Could Have (있으면 좋음)
```
- [ x] 태그 기능
- [ ] 통계 대시보드
- [ ]
```

#### Won't Have (이번 버전 제외)
```
- [ ] 팀 협업 기능
- [ ]
```
